# LinkedIn Organic Content Strategy for B2B SaaS — Research

*Work in progress.*

This repository collects research on LinkedIn organic content strategy for B2B SaaS,
based on the recent public content of 10 selected experts in this space.

## Repository structure

- [`research/sources.md`](research/sources.md) — the 10 experts researched, with links and notes on why each was chosen
- [`research/linkedin-posts/`](research/linkedin-posts/) — recent LinkedIn posts from each expert, collected manually
- [`research/youtube-transcripts/`](research/youtube-transcripts/) — transcripts of their YouTube/podcast appearances, collected via the Supadata API
- [`research/other/`](research/other/) — newsletters, articles, and a playbook outline distilled from the research

More details (why this topic, why these experts, and the playbook outline) will be added as the research progresses.
